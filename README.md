# ✈️ Flight Price Tracker

A smart flight tracking system that simulates real-time airline price fluctuations and alerts users about the best time to book.

---

## 🚀 Features

- 📊 Real-time price simulation using historical data
- 🔔 Price drop / increase alerts
- 💾 Price history tracking (JSON storage)
- 🤖 Basic price prediction system
- 🧠 Intelligent input normalization (typo tolerant)

---

## 🛠️ Tech Stack

- Python
- Pandas
- Random simulation models
- JSON storage
- OOP architecture
